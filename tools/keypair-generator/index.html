Private Key & Public Key Generator

A tool to generate public key and private key pair. Supported Elliptic-curve-based key pair (ECDSA or ECDH) and RSA-based key pair (RSASSA-PKCS1-v1_5, RSA-PSS or RSA-OAEP).

Generating any RSA-based key pair: that is, when the algorithm is identified as any of RSASSA-PKCS1-v1_5, RSA-PSS, or RSA-OAEP.
