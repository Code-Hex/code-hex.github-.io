White-MobileTerminal

JailBreak 済み iOS 端末向けの Terminal アプリである Mobileterminal というプロジェクトを fork して主にデザイン面に力を注いだプロジェクト. White-Terminal では 前作にはないプロンプトの背景色, フォントの変更, フォントカラーの変更, カーソルカラーの変更などができるようになった. 更に, function key や control key, tab, esc が使えるようになったため, シグナルの送信も楽になった. 途中から OCR の機能をオープンソースである Tesseract を用いて使えるようにした. 容量が大きすぎて配布元のサーバーにアップロードしても反映されなくなったため開発が停止しているが, Swift を使用してモダンな形でリメイクしようと構想中. 現在総合ダウンロード数は 50000 回を超えている

Pget

2016 年の Hackers Champloo の開発合宿で仕上げようとした作品. 開発合宿の期間中に完成までに至らなかったため, カンファレンスの部での LT についてはなんとかごまかした. このコマンドは Range access が可能なサーバーに対してコマンド側で指定した数分の goroutine を作成して並列処理でダウンロードを行う. 最終的に, 分割されたファイルがダウンロードされ, それらを結合するといったクライアントである. 現在は構想した段階までは完成してはいないが, ダウンロードを行う前にコンピュータのディスクの容量は足りているか, 対象の URL がリダイレクトを行い, その最後の URL が Range access に対応しているかなどの確認や, 並列ダウンロード中に何かしらのエラーが発生した場合, ダウンロードのために作成された goroutine が安全に終了するように工夫されている.
